# Nitroapp (iserv-api)
An API for getting exercise-information from IServ


## Get Started
> 1. Install requirements
>`pip install -r .\requirements.txt`

>2. Edit `.env.example` and fill in your credentials

>3. Run the script
>`py .\src\main.py`
